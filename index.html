MODEL RULES

World Conditions + Bonus Systems

Base Q

Base Q is the primary decision layer. It scores gather, move, rest, socialize, and observe using body state, moon resources, memory, confidence, reward history, drought, river access, and personality bias. Optional tools can nudge the scores, but Base Q remains the main brain.

Agent Profiles

Maya is the risk-taking explorer, Pollock is more cautious and observant, and Rembrandt is more social. These profiles bias behavior without fully overriding the learned Q values.

Drought Conditions

Drought is a temporary moon condition. During drought, food and water still regenerate, but regeneration is reduced to about 58.75% of normal. River moons become more valuable during drought because river access partly offsets the dry penalty.

Resource Regeneration

Each moon stores food and water up to a cap of 200. Food returns through fertility and the food regeneration multiplier. Water returns through aquifer strength, the resource regeneration multiplier, and any river buff.

River Bonus

River access improves water recovery. Normal river benefit is 1.15. During drought, the river drought buff rises to 1.65, making river-adjacent moons more survivable without making drought harmless.

Bonus Systems

Bountiful Harvest is a rare long-cycle event, roughly once every 25 simulated years with variation. It lasts 2 days and gives living agents a +5 health cushion per harvest tick. Social Compensation gives agents a small starting social cushion. Respawn keeps dead agents inactive for 5 days before they return with warning marks intact.

Agent Lexicon Correction

Light Influence is now weaker and cleaner. It blocks repeated journal-template words such as gathered, result, and route, favors condition words such as drought, river, water, pain, warning, supply, body, thirst, and hunger, and keeps semantic nudges small so the lexicon cannot overpower Base Q.

PROJECT README

Microsmos

Microsmos is a micro AI observatory for learning agents: a retro, browser-based simulation focused on journal language, Q patterns, memory, stress, and divergent behavior signals.

Each moon runs as a dynamic grid-based environment where agents respond to resources, memory, drought, and social signals.

This project was designed by Jesse Glick, a new learner in software design and computer science. The AI model was first implemented in the earlier app Jungle Moon; this version refines the concept into a cleaner educational research tool.

System design and direction are by Jesse Glick. All material and implementation are AI-assisted.

Free to use for educational and scientific purposes with credit.

The simulation loads in a ready state and begins only when Start is pressed. Reset returns the run to Day 1 without auto-starting.

Model options

Current Model: Q + Journal Display keeps the journal vocabulary as an observation layer. Words are highlighted as novel language, but they do not change the agent's future choices.

Agent Lexicon can record word-condition links from journal language. In Display Only mode it watches without changing behavior; in Light Influence mode it gives tiny action nudges while the Q-learning system remains the main decision engine.

World Conditions + Bonus Systems explains drought, resource regeneration, river buffs, bountiful harvest, respawn timing, Base Q, and the agent profiles.
